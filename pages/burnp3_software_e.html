Important Installation Instructions

  • Install Prometheus 2021.12.03 (BurnP3 version must match the Prometheus Version or you will get an annoying messages on startup - but it will still work) and all associated software prerequistes on a 64-bit Windows computer that meets or exceeds minimum system requirements and perform a system restart.
  • Log on to your computer as a system administrator.
  • Uninstall any previous version of the Burn-P3 application from your computer.
  • Close all open applications and run BurnP3_2021.08.00_English.msi [MSI - 5.34 MB]. A Setup Wizard will guide you through the installation.
