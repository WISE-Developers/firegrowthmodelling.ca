Important Installation Instructions

  • Install SyncroSim SyncroSim is the main tool that coordinates fire growth modelling for Burn-P3+.

If you use the Conda option in SyncroSim, you do not need to do the following steps.

  • Install R
    R is the underpinning of the Burn-P3+ methodology. SyncroSim leverages RSyncroSim to perform the modelling within the SyncroSim software.
  • Install Prometheus 2021.12.03
    Prometheus is a requirement of the tool if you intend to run without using Conda. If you use the Conda option, you do not need to have Prometheus installed.
