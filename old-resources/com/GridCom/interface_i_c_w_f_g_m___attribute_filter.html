Detailed Description

Interface ICWFGM AttributeFilter.

Attribute Filter is a COM Interface, part of GridCom.

Member Function Documentation

ICWFGM_AttributeFilter::Clone ( [out] ICWFGM_AttributeFilter **  newFilter)

Creates a new AttributeFilter object with all the same properties of the object being called, returns a handle to the new object in ‘newFilter’. No data is shared between these two objects, an exact copy is created. However, handles to ICWFGM_Fuel, etc. objects are duplicated, not the ICWFGM_Fuel objects themselves.

Property ICWFGM_AttributeFilter::OptionKey
getsetproperty

Gets and Sets Option Key. If the option key is set to (USHORT)-1, then, via its ICWFGM_GridEngine interface, this object acts like a replacement fuel grid. If the option key is set to any other value then, via its ICWFGM_GridEngine interface, it will respond to gridded GetAttributeData and GetAttributeDataArray method invocations when this key matches the optionKey parameter to these calls.
